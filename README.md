# Calendar Python

A simple CLI-based calendar application built with Python.

## Features

- Interactive command-line interface
- Built using Python's standard library
- Lightweight and easy to use

## Requirements

- Python 3.x

## Installation

Clone the repository:

```bash
git clone https://github.com/teguhsuryaro/calendar-python.git
cd calendar-python
```

## Usage

Run the main application:

```bash
python calendarApp.py
```

## Project Structure

```
calendar-python/
├── calendarApp.py    # Main application file
├── tools.py          # Helper utilities
└── .gitignore
```
